- # Simple Push-up Counter

  A minimal, single-page push-up counter for touch-based use.

  Tap the screen with your nose after each push-up to increase the count.

  Best used on mobile devices. Desktop usage is not adapted.

  ## Usage

  1. Visit https://simple-push-up-counter.vercel.app/
  2. Start your push-ups
  3. Tap the screen to count

  ## Notes

  - No data is stored
  - No analytics
